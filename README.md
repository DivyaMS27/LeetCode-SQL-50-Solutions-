# LeetCode SQL 50 Solutions

This repository contains my solutions to the **LeetCode SQL 50** problems.  
It covers topics like SELECT, JOIN, Aggregation, Sorting/Grouping, Window Functions, Subqueries, and Advanced SQL.

## 📂 Repository Structure
- `Select/` → Basic SELECT queries
- `Joins/` → INNER JOIN, LEFT JOIN, etc.
- `Aggregation/` → GROUP BY, HAVING
- `Sorting_Grouping/` → ORDER BY, GROUP BY
- `Advanced_Select_Joins/` → Complex joins and advanced selects
- `Subqueries/` → Nested queries
- `String_Regex/` → String functions, regex, clauses
